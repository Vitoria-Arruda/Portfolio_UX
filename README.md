# Wireframe — Aprovação de Documentos | Prothera

## Sobre o projeto
Wireframe mobile desenvolvido como teste prático para processo seletivo da Prothera Tecnologia.

## Desafio
Criar uma tela de aprovação de documento que permitisse ao usuário visualizar, acessar e aprovar ou reprovar documentos.

## Solução
Fluxo completo de 4 telas: Lista de documentos → Resumo → Visualização → Resultado
